# AI Environment Setup

## Overview

This repository documents the setup process of the required tools for this project, including Cursor IDE, Claude Code, Codex, Git, and GitHub.

## Tools Installed

* Cursor IDE
* Claude Code Extension
* Codex Extension
* Git
* GitHub

## Steps Completed

1. Installed Cursor IDE.
2. Installed the Claude Code extension in Cursor and logged in.
3. Installed the Codex extension in Cursor and logged in.
4. Created a public GitHub repository.
5. Opened the repository in Cursor.
6. Created this README file to document the process.
7. Committed and pushed the changes to GitHub.

## Issues Encountered

### Git Configuration

**Issue:** Git required user information before creating commits.

**Solution:** Configured Git username and email using Git commands.

### Extension Login

**Issue:** Initial setup required signing in to the installed extensions.

**Solution:** Followed the authentication steps provided by each extension.
